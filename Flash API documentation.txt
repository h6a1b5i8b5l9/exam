Пример использования:

	Flash::setMessage('test', 'Hi');
	echo Flash::getMessage('test');

описание:
	
	Компонент Flash это основанный на сессиях компонент, для вывода флеш сообщений/уведомлений.
	Компонент Flash содержит 2 открытых статичных функции для использования: setMessage(...), getMessage(...).
	Для использования этих функций надо подключить класс Flash. Так как методы работают через сессии, их подключение реализованно внутри самих методов.

	Функция setMessage(...) реализует запись флеш сообщения. Она принимает 2 параметра (string $name, string $message) :NoReturn.
	-Первый: $name это имя под которым в сессию будет записано сообщение;
	-Второй: $message это само сообщение, которое будет записано.
	
	Функция getMessage(...) реализует запись флеш сообщения. Она принимает 1 параметр (string $name) :string.
	- $name это имя в сессии под которым было записано сообщение;
	После вывода сообщение оно удаляется из сессии.
